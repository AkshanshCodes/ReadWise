# 📚 ReadWise – Smart Reading Assistant

## 📖 Project Description

ReadWise is a smart reading assistant web application that helps users discover, explore, and manage books efficiently. Users can search for books, view details such as title, author, and cover, and build a personalized reading library.

The application goes beyond basic search by enabling users to filter, sort, and organize books while also providing simple recommendation logic based on user preferences.

---

## 🎯 Purpose

The purpose of this project is to demonstrate the use of JavaScript, API integration, and dynamic UI development by building a real-world application that improves the user’s reading experience.

---

## 🔌 API Used

* Open Library API
  Used to fetch book data such as title, author, and cover images.
  https://openlibrary.org/search.json?q=${query}

---

## ✨ Features

### 🔍 Search

* Users can search for books by title or keyword.

### 📊 Filtering & Sorting

* Filter books based on category/genre (planned)
* Sort books alphabetically or by relevance (planned)

### 📚 Book Display

* Displays book title, author, and cover image dynamically.

### ❤️ Personal Library (Planned)

* Users can save books to their personal collection.
* Books can be marked as:

  * Want to Read
  * Reading
  * Completed

### 🧠 Recommendation System (Planned)

* Suggest books based on user preferences and previously saved items.

### 📱 Responsive Design

* Works across mobile, tablet, and desktop devices.
